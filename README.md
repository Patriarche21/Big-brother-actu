# BigBro Project

## Description
Application de surveillance et d'analyse de contenu avec :
- Backend Node.js
- Frontend React/Vite

## Installation

### Pr√©requis
- Node.js v18+
- npm v9+

### Backend
```bash
cd backend
npm install
```

### Frontend
```bash
cd frontend
npm install
```

## Configuration
Copier le fichier `.env.example` en `.env` et remplir les variables d'environnement.

## Lancement

### Backend
```bash
cd backend
npm start
```

### Frontend
```bash
cd frontend
npm run dev
```

## Structure du projet
- `backend/` : Serveur Node.js
- `frontend/` : Application React/Vite
